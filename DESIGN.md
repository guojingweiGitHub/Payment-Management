# DESIGN.md

## 气质与意象
简洁实用的项目管理界面，类似企业内部管理系统的风格，强调数据的清晰展示和高效操作。

## Design Tokens

### 色彩
- 主色：中性灰调（#f5f5f5 背景）
- 强调色：蓝色系用于主要操作按钮
- 状态色：
  - 未缴费：橙色/红色警示
  - 已缴费：绿色成功
  - 到期：红色高亮

### 字体
- 中文优先：系统默认字体

### 布局
- 表格为主的数据展示
- 顶部筛选栏
- 弹窗式新增/编辑操作
- 卡片式详情展示

### 组件规范
- 使用 shadcn/ui 组件库
- 表格：DataTable 支持分页、排序、筛选
- 表单：Form + Input/Select/DatePicker 组合
- 弹窗：Dialog 组件
